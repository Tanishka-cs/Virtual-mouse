# 🖱️ Virtual Mouse using Hand Gestures

Control your computer without touching a mouse!  
This project uses **Computer Vision** to track your hand and convert gestures into mouse actions.

---

## 🚀 Features

- ✋ Cursor movement using index finger  
- 👌 Click using pinch gesture (thumb + index)  
- 📜 Scroll using middle finger  
- 📸 Screenshot using hand gesture  
- ⚡ Smooth cursor movement  
- 🎥 Real-time hand tracking  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- MediaPipe  
- PyAutoGUI  
- NumPy
